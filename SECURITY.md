# 安全策略

## 支持版本

当前只对最新发布的 minor 版本提供安全修复。旧版本如果存在安全或稳定性问题，会通过 patch 版本给出迁移建议。

| 版本 | 支持状态 |
| --- | --- |
| `0.7.x` | 支持 |
| `<0.7.0` | 仅建议升级 |

## 报告安全问题

如果你发现可能导致崩溃、拒绝服务、越权文件访问或供应链风险的问题，请通过 GitHub 私密安全报告提交，不要先公开 issue。

报告时请包含：

- 受影响版本。
- Flutter / Dart 版本。
- Android 或 iOS 系统版本。
- 可复现的最小输入，尤其是图片格式、尺寸和文件大小。
- 崩溃日志、异常堆栈或复现步骤。

## 处理原则

- 确认可复现后，会优先准备 patch 版本。
- 涉及恶意图片输入导致的内存或 CPU 风险，会同步补充 benchmark 或回归测试。
- 发布后会在 `CHANGELOG.md` 记录修复范围和迁移建议。
