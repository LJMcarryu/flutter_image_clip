# 平台与 CI 矩阵

## 支持范围

| 维度 | 当前策略 |
| --- | --- |
| Flutter | `>=3.38.1`，CI 同时覆盖最低支持版本和最新 stable |
| Dart | `>=3.10.0 <4.0.0` |
| 平台 | Android、iOS |
| 公开入口 | 只承诺 `package:flutter_image_clip/flutter_image_clip.dart` 导出的 API |
| 发布版本 | 破坏性变更升主版本，新增能力升次版本，兼容修复升 patch |

## 原生解码能力

| 平台 | 解码入口 | 覆盖能力 | 失败映射 |
| --- | --- | --- | --- |
| Android | `BitmapFactory` / `ImageDecoder` | PNG、JPEG、WebP、系统支持的 HEIC/HEIF，支持 sampled preview decode | `unsupported_format`、`invalid_args`、`decode_failed` |
| iOS | `CGImageSource` / `UIImage` | PNG、JPEG、HEIC/HEIF，支持 `CGImageSourceCreateThumbnailAtIndex` 预览解码 | `unsupported_format`、`invalid_args`、`encode_failed`、`decode_failed` |
| Dart fallback | `package:image` | PNG、JPEG、GIF、WebP 等纯 Dart 可解码格式 | `ImageClipUnsupportedFormatException`、`ImageClipDecodeException` |

`ImageClipPlatformDecodeAdapter` 会把平台侧错误统一映射到 Dart typed exception：

- `unsupported_format` -> `ImageClipUnsupportedFormatException`
- `invalid_args` / `platform_unavailable` -> `ImageClipPlatformException`
- `encode_failed` / `decode_failed` -> `ImageClipDecodeException`

## CI 作业

| 作业 | 运行环境 | 内容 |
| --- | --- | --- |
| `Flutter checks` | Ubuntu，Flutter `3.38.1` 和 stable | format、analyze、unit/widget test、API snapshot、benchmark baseline、dartdoc、pub dry-run、Android example debug build |
| `iOS example build` | macOS stable | 生成 iOS example 工程并执行 `flutter build ios --debug --no-codesign` |
| `Android integration test` | Ubuntu emulator，Flutter `3.38.1` 和 stable | 在 example App 中运行 `integration_test`，覆盖编辑器流程和原生 sampled decode |
| `iOS integration test` | macOS simulator stable | 在 iOS simulator 中运行 `integration_test`，覆盖编辑器流程和原生 sampled decode |
| `Release checks` | tag `v*` 触发 | 重复发布前 format、analyze、test、API、公开 API 文档、benchmark、dartdoc 和 pub dry-run；全部通过且启用 `PUB_DEV_AUTOMATED_PUBLISHING` 后进入 pub.dev 发布 job |
| `Security checks` | pull request | dependency review，阻止高风险依赖变更进入主分支 |

## 发布前本地校验

```sh
flutter pub get
dart format lib test benchmark tool example/lib example/integration_test
flutter analyze
flutter test
dart run tool/check_api_snapshot.dart
dart run tool/check_public_api_docs.dart
dart run benchmark/image_processor_benchmark.dart --check benchmark/baseline.json
dart doc --output doc/api
dart pub publish --dry-run
```

发布前应确认 `CHANGELOG.md` 有当前 `pubspec.yaml` 版本条目，`tool/api_snapshot.json` 已随公开 API 变化更新，`dart pub publish --dry-run` 没有新增错误。自动发布需要先在 pub.dev Admin 中启用 GitHub Actions OIDC，tag pattern 使用 `v{{version}}`。

## 设备补测建议

CI 覆盖 emulator/simulator，但正式发布前仍建议在真实设备补测：

- Android 10、Android 13+：JPEG、PNG、透明 PNG、大图 sampled decode、HEIC/HEIF。
- iOS 15、iOS 最新稳定版：相册 HEIC、JPEG EXIF orientation、透明 PNG、低内存场景。
- 慢速设备：连续旋转、缩放、保存、取消任务，确认 UI isolate 不被大图处理阻塞。
