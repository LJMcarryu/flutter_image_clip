# 接入配方

这份文档按常见业务场景给出推荐接入方式。示例默认从 `package:flutter_image_clip/flutter_image_clip.dart` 导入公开 API。

## 头像裁剪

头像通常需要固定正方形、JPEG 输出和较小预览图：

```dart
final result = await showImageClipEditor(
  context,
  imagePath: avatarPath,
  imageLabel: 'avatar.jpg',
  initialAspectRatio: ImageClipAspectRatio.square,
  aspectRatios: const [ImageClipAspectRatio.square],
  outputSettings: const ImageClipOutputSettings.jpeg(jpegQuality: 88),
  previewDecodeSettings: const ImageClipDecodeSettings.preview(
    targetLongSide: 1200,
  ),
);
```

## 封面图裁剪

封面图建议保留多个业务比例，保存时输出 JPEG：

```dart
ImageClipEditor(
  initialImagePath: coverPath,
  initialImageLabel: 'cover.jpg',
  initialAspectRatio: ImageClipAspectRatio.widescreen,
  aspectRatios: const [
    ImageClipAspectRatio.widescreen,
    ImageClipAspectRatio.ratio16x10,
    ImageClipAspectRatio(label: '3:1', width: 3, height: 1),
  ],
  outputSettings: const ImageClipOutputSettings.jpeg(jpegQuality: 90),
  previewDecodeSettings: const ImageClipDecodeSettings.preview(
    targetLongSide: 1600,
  ),
  showResultPage: false,
  onResult: (result) {
    uploadCover(result.cropped.bytes);
  },
);
```

## 配合 image_picker

相册 SDK 返回文件路径时，优先把路径直接交给编辑器；只有路径不可用、网络图或内存图才退回 `XFile.readAsBytes()`：

```dart
final picked = await ImagePicker().pickImage(source: ImageSource.gallery);
if (picked == null) {
  return;
}

final result = await showImageClipEditor(
  context,
  imagePath: picked.path,
  imageLabel: picked.name,
  previewDecodeSettings: const ImageClipDecodeSettings.preview(
    targetLongSide: 1600,
  ),
);
```

如果没有稳定文件路径，再传 bytes：

```dart
final bytes = await picked.readAsBytes();
final result = await showImageClipEditor(
  context,
  imageBytes: bytes,
  imageLabel: picked.name,
);
```

如果不需要交互裁剪，也可以直接处理文件路径，避免业务层提前读入大图：

```dart
final processor = ImageProcessor();
final result = await processor.processFile(
  picked.path,
  steps: const [
    ImageClipPipelineStep.resizeLongSide(1600),
  ],
  outputSettings: const ImageClipOutputSettings.jpeg(jpegQuality: 88),
);
```

## HEIC 和 HEIF

纯 Dart 解码无法处理 HEIC/HEIF。`showImageClipEditor` 已默认启用内置平台适配器；直接使用 `ImageProcessor` 时建议显式传入：

```dart
final processor = ImageProcessor(
  decodeAdapter: const ImageClipPlatformDecodeAdapter(),
);

final preview = await processor.decodePreviewBytes(
  bytes,
  label: 'camera.heic',
  targetLongSide: 1600,
);
```

如果业务相册 SDK 已经把 HEIC 转成 JPEG/PNG，可以继续使用默认 `ImageProcessor`。

本地文件保存只包含区域裁剪、90 度倍数旋转和翻转且输出为 JPEG 时，内置平台适配器会优先尝试原生 `cropFile`，失败时自动回退 Dart isolate；PNG 输出继续走 Dart 管线以保留 `pngLevel` 行为：

```dart
final result = await processor.processFile(
  picked.path,
  steps: const [
    ImageClipPipelineStep.cropRegion(
      CropRegion(x: 120, y: 80, width: 640, height: 480, cornerRadius: 0),
    ),
    ImageClipPipelineStep.rotate(),
  ],
  outputSettings: const ImageClipOutputSettings.jpeg(jpegQuality: 88),
);
```

## 大图保护

对相机原图、长截图和服务端未知来源图片，建议明确设置处理上限：

```dart
final processor = ImageProcessor(
  processingSettings: const ImageClipProcessingSettings(
    maxInputPixels: 48000000,
    maxOutputPixels: 16000000,
    autoDownscale: true,
  ),
);
```

`maxInputPixels` 用于拒绝异常大图，`maxOutputPixels` 用于自动压缩输出结果，避免业务上传或预览时出现内存峰值。

## 自定义 UI

业务需要自己的按钮、导航或表单结构时，使用 `ImageClipEditorController` 驱动编辑器：

```dart
final controller = ImageClipEditorController();

ImageClipEditor(
  controller: controller,
  loadSampleOnStart: false,
  showResultPage: false,
  onResult: (result) {
    submit(result.cropped.bytes);
  },
);

await controller.loadImage(bytes, label: 'profile.jpg');
await controller.rotateRight();
await controller.flipHorizontal();
final result = await controller.crop();
```

## 异常处理模板

```dart
try {
  final result = await processor.decodeBytes(bytes, label: label);
  return result;
} on ImageClipUnsupportedFormatException catch (error) {
  showMessage('当前图片格式暂不支持：${error.format}');
} on ImageClipImageTooLargeException catch (error) {
  showMessage('图片过大：${error.width}x${error.height}');
} on ImageClipPlatformException catch (error) {
  showMessage('系统图片解码不可用：${error.message}');
} on ImageClipDecodeException catch (error) {
  showMessage('图片解码失败：${error.message}');
}
```
